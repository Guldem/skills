# Skills

A collection of agent skills for Dart and Flutter development.

| Skill | Description |
| --- | --- |
| [dart_dot_short_hands](skills/dart_dot_short_hands) | Use Dart's dot-shorthand syntax (`.foo`) to access enum values, static members, and constructors without repeating the type name when it's inferable from context. |
| [dart_enhanced_enums](skills/dart_enhanced_enums) | Declare and use Dart enhanced enums — enums with fields, constructors, methods, getters, and interfaces. |
| [flutter_slivers](skills/flutter_slivers) | Use Flutter's sliver widgets (`CustomScrollView`, `SliverAppBar`, `SliverList`, `SliverGrid`, `SliverPersistentHeader`, etc.) to achieve fancy, high-performance scrolling effects. |
